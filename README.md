## What E-commerce can do! 
### MySql database that runs the backend of an E-commerce site! 

## How to Install!
npm init

npm install mysql2

npm install sequelize

npm install dotenv

## How to Use!
### In your terminal run the following steps!
mysql -u root -p
enter password  

source db/schema.sql  
quit   

npm run seed  

npm start
